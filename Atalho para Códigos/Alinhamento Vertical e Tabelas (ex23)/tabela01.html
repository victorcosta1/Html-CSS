Tabelas

Para Fazermos tabelas em HTML utilizamos a tag table dentro dela utilizamos tr e detro da mesma td

Sendo assim a respectiva ordem hierarquica:

table: Tag main todas as outras vão dentro dela
tr: Tr = Table row: Representa a linha da tabela
td: td = Table data: Representa os dados contidos na linha

Sendo assim a tabela ficara desta forma:

A1 B1 C1
A2 B2 C2
A3 B3 C3
